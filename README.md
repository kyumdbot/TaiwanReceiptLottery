# 發票對獎相機 (macOS App)

- 使用 AV Foundation 存取 Webcam。
- 使用 Vision Framework 檢測文字。


## 說明

1. 可檢測傳統發票上的發票號碼，並且對獎。
2. 中獎的號碼會以紅色字體顯示，沒中獎的號碼以藍色字體顯示。
